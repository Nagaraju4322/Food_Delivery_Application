# Food_Delivery_Application


Food Delivery Application – Java Full Stack Project
Project Overview
The Food Delivery Application is a full-stack web application developed using Java and web
technologies. It allows users to browse restaurants, view menus, add food items to a cart, and place
orders seamlessly. The application follows the DAO (Data Access Object) design pattern to ensure
clean architecture and maintainability. It is deployed on Apache Tomcat Server and uses MySQL
for persistent data storage.

# Features
• User Registration and Login
• View Available Restaurants
• Browse Menu Items
• Add and Remove Items from Cart
• Order Summary and Checkout
• MySQL Database Integration
• DAO Design Pattern Implementation
• Responsive UI using HTML and CSS
# Technologies Used
Backend: Java, Servlets, JDBC, DAO Design Pattern
Frontend: HTML5, CSS3
Database: MySQL
Server: Apache Tomcat

# Project Architecture
The application follows the MVC architecture: Model – Java classes and DAO implementations
View – HTML and CSS pages
Controller – Servlets handling requests and responses

# Modules
• User Authentication Module
• Restaurant Management Module
• Menu Management Module
• Cart Management Module
• Order Processing Module
